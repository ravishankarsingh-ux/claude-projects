Adding another project

  1. Create a new top-level folder named after the project — e.g. Acme Dental Clinic/ — sitting next to Mom Grandridge School/, not inside it.
  2. Drop in a self-contained index.html (plus its own css/, js/, assets/) — every project is independent, so nothing here can break another site.
  3. Add one row to this page — copy a <a class="proj"> block above, point the link at the new folder, and update the name/description/tags. Ask me to do this and I'll wire it up whenever you hand off new project code.
